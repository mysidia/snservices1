Query Messages

System Queries

  • QUERY SYSTEM UPTIME

Object Queries

  • QUERY OBJECT AKILL|AHURT|IGNORE LIST

  • QUERY OBJECT RNICK nick ISREG

    Results in, for instance:

    OK QUERY OBJECT RNICK=nickname ISREG=TRUE - Nickname is registered.

    OR

    OK QUERY OBJECT RNICK=nickname ISREG=FALSE - Nickname is not registered.

  • QUERY OBJECT RNICK nick ISBYPASS

    Example response:

     
    OK QUERY OBJECT RNICK=nick ISBYPASS=TRUE
    
						
    ERR-BADTARGET QUERY OBJECT RNICK=nickname - NOT_REGISTERED

  • QUERY OBJECT RNICK nick REG-TIME

    Sample result:

    OK QUERY OBJECT RNICK=nickname REG-TIME 861620971

  • QUERY OBJECT RNICK nick LAST-TIME

    Sample result: OK QUERY OBJECT RNICK=nickname LAST-TIME 1067654340

  • QUERY OBJECT RNICK nick MARK

    Sample result: OK QUERY OBJECT RNICK=Mysidia MARK NO

    OK QUERY OBJECT RNICK=Mysidia MARK YES

  • QUERY OBJECT RNICK nick LAST-HOST

    Sample result:

    OK QUERY OBJECT RNICK=nickname LAST-HOST asdf.com.hmsk
    
						
    ERR-NOPRIV QUERY OBJECT RNICK=nickname LAST-HOST - User WWW/test Not authorized to query RNICK LAST-HOST.

  • QUERY OBJECT RNICK nick NUM-MASKS

  • QUERY OBJECT RNICK nick ACCESS-*

  • QUERY OBJECT RNICK nick ACCESS-number

  • QUERY OBJECT RNICK nick URL

  • QUERY OBJECT RNICK nick EMAIL

  • QUERY OBJECT RNICK nick LAST-HOST-UNMASK

  • QUERY OBJECT RNICK nick FLAGS

  • QUERY OBJECT RNICK nick OPFLAGS

  • QUERY OBJECT RNICK nick BADPWS

  • QUERY OBJECT RNICK nick TIMEREG

  • QUERY OBJECT RNICK nick IS-READTIME

  • QUERY OBJECT RNICK nick ACC
